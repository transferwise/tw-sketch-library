# tw-sketch-library
Sketch library of TW UI components for use with Invision's craft sketch plugin

### Instructions

1. Install Sketch - https://www.sketchapp.com/
2. Get the plugin - https://www.invisionapp.com/craft
3. Download this repo
4. Toggle Craft open in Sketch
5. Import the TW library into sketch through the plugin
6. Shut up Derek
